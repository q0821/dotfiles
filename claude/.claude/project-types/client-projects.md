# 客戶專案管理指導

## 專案階段

### 1. 前期溝通（Pre-Sales）

#### 初步接洽
- 了解客戶需求和預算範圍
- 評估專案可行性和時程
- 確認決策者和窗口
- 詢問參考網站（喜歡的風格）

#### 需求確認清單
```markdown
## 基本資訊
- [ ] 網站類型（形象官網/電商/部落格/客製化）
- [ ] 目標受眾
- [ ] 主要功能需求
- [ ] 預期上線時間

## 技術需求
- [ ] 網域/主機狀況（新購/現有）
- [ ] 多語言需求
- [ ] 整合需求（金流/ERP/CRM）
- [ ] 特殊功能（會員系統/預約/線上表單）

## 內容與素材
- [ ] Logo 和企業識別（有/無/需設計）
- [ ] 文案內容（客戶提供/需協助撰寫）
- [ ] 圖片素材（提供/需攝影/使用圖庫）
- [ ] 影片素材

## 維護與支援
- [ ] 需要維護合約
- [ ] 教育訓練需求
- [ ] 後續支援範圍
```

### 2. 報價與合約

#### 報價單內容

```markdown
## 專案報價單

### 一、專案範圍
1. 網站設計
   - 首頁設計
   - 內頁設計（X 款）
   - RWD 響應式設計

2. 功能開發
   - 內容管理系統（WordPress）
   - 聯絡表單
   - [其他功能項目]

3. 技術服務
   - 網域申請/轉移
   - 主機租用（第一年）
   - SSL 憑證設定

### 二、交付項目
- [ ] 完整網站原始碼
- [ ] 後台操作手冊
- [ ] 教育訓練（X 小時）
- [ ] 3 個月免費維護

### 三、付款方式
- 簽約時：30%
- 設計稿確認：30%
- 網站上線：40%

### 四、時程規劃
- 簽約後 2 週：設計稿交付
- 設計稿確認後 3 週：開發完成
- 測試確認後 1 週：正式上線

總計：5-6 週
```

#### 合約注意事項

**必須明確規範**：
- 修改次數上限（例：設計稿 2 次修改）
- 驗收標準
- 超出範圍的額外收費方式
- 素材提供時程（客戶延遲則時程順延）
- 智慧財產權歸屬

**保護條款**：
- 客戶提供素材的合法性由客戶負責
- 第三方服務（金流、API）費用另計
- 客戶修改原始碼後不保固

### 3. 設計階段

#### 設計稿交付

**首頁設計稿包含**：
- 桌面版（1920px）
- 平板版（768px）
- 手機版（375px）

**內頁範例**：
- 關於我們
- 服務項目
- 最新消息（列表頁 + 內頁）
- 聯絡我們

#### 設計稿回饋管理

**統一收集回饋**：
```
請客戶將所有修改意見整理成文件，一次提供。
避免零散的訊息轟炸。
```

**範例回饋格式**：
```markdown
## 設計稿修正意見 - 第 1 版

### 首頁
1. 主視覺圖片希望更換為 XXX
2. 標題文字改為「XXX」
3. 按鈕顏色希望改為品牌色

### 關於我們頁
1. 團隊照片位置調整
2. 文字段落重新編排
```

### 4. 開發階段

#### Staging 環境設定

**提供測試網址**：
```
https://staging.clientdomain.com
帳號：client_preview
密碼：[臨時密碼]
```

**設定 robots.txt 阻擋搜尋引擎**：
```
User-agent: *
Disallow: /
```

#### 進度回報

**每週更新**（Email 或專案管理工具）：
```markdown
## 專案進度更新 - Week 3

### 本週完成
- [x] 首頁功能開發完成
- [x] 最新消息系統建置

### 下週計劃
- [ ] 聯絡表單整合
- [ ] 產品頁面開發

### 需要客戶配合
- 請提供公司簡介文案
- 確認產品分類架構
```

### 5. 測試與驗收

#### 測試清單

```markdown
## 網站測試檢查表

### 功能測試
- [ ] 所有連結正常運作
- [ ] 表單送出成功並收到通知
- [ ] 搜尋功能正常
- [ ] 會員登入/註冊（如有）
- [ ] 金流測試（如有）

### 瀏覽器相容性
- [ ] Chrome（最新版）
- [ ] Safari（最新版）
- [ ] Firefox（最新版）
- [ ] Edge（最新版）
- [ ] 行動裝置瀏覽器

### 裝置測試
- [ ] 桌面（1920px, 1366px）
- [ ] 平板（768px）
- [ ] 手機（375px, 414px）

### 效能測試
- [ ] PageSpeed Insights > 85 分
- [ ] 首頁載入 < 3 秒

### SEO 基礎
- [ ] Meta Title 和 Description 設定
- [ ] H1-H6 標籤結構正確
- [ ] 圖片 Alt 屬性
- [ ] Sitemap 產生並提交
```

#### 客戶驗收

**驗收會議議程**：
1. 功能逐一展示
2. 記錄客戶提出的問題
3. 確認修正項目和時程
4. 確認上線日期

**驗收單範例**：
```markdown
## 網站驗收確認單

專案名稱：XXX 官方網站
驗收日期：YYYY-MM-DD

### 交付項目確認
- [ ] 網站功能符合合約規範
- [ ] 設計符合確認的設計稿
- [ ] 測試項目全數通過
- [ ] 後台操作手冊已交付

### 已知問題（預計上線後修正）
1. [如有]

客戶簽名：_____________
日期：_____________
```

### 6. 上線階段

#### 上線前檢查

```markdown
## 上線前最終檢查

### 內容確認
- [ ] 刪除所有測試內容
- [ ] 聯絡資訊正確（電話、Email、地址）
- [ ] 社群媒體連結正確
- [ ] 隱私權政策頁面

### 技術設定
- [ ] DNS 設定正確
- [ ] SSL 憑證安裝
- [ ] 強制 HTTPS 重新導向
- [ ] 移除 staging 環境 robots.txt
- [ ] 正式環境 robots.txt 設定
- [ ] Google Analytics 追蹤碼
- [ ] Google Search Console 驗證
- [ ] Facebook Pixel（如需要）

### 安全性
- [ ] 變更管理員帳號（不使用 admin）
- [ ] 強式密碼設定
- [ ] 限制登入嘗試次數
- [ ] 關閉 WP_DEBUG
- [ ] 備份機制設定

### 效能優化
- [ ] 快取外掛啟用
- [ ] 圖片壓縮
- [ ] CSS/JS 壓縮
- [ ] CDN 設定（如使用 Cloudflare）
```

#### 上線流程

**建議時間**：週間非尖峰時段（避免週五下午、週末）

**步驟**：
1. 備份 Staging 環境資料庫和檔案
2. DNS TTL 調降至 300 秒（提前 24 小時）
3. 上傳檔案到正式環境
4. 匯入資料庫並修改網址
5. 測試所有功能
6. 修改 DNS 指向
7. 等待 DNS 生效（通常 1-4 小時）
8. 確認正式網址正常運作

#### 教育訓練內容（2 小時）

**基礎操作**（1 小時）：
- WordPress 後台導覽
- 新增/編輯頁面
- 新增/編輯文章
- 媒體庫管理
- 選單管理

**進階功能**（30 分鐘）：
- 特定外掛操作（如表單設定）
- SEO 基礎設定
- 備份與還原

**注意事項**（30 分鐘）：
- 不可修改的部分
- 如何申請技術支援
- Q&A

### 處理困難客戶

#### 案例 1：無止盡的修改要求

**策略**：
- 明確指出已超出合約的修改次數
- 提供額外修改的報價單
- 保持專業但堅定

**話術**：
```
「根據合約，設計稿包含 2 次修改機會，目前已完成。
若需要進一步調整，我們很樂意協助，
額外修改費用為每次 NT$ X,XXX。」
```

#### 案例 2：客戶要求不合理功能

**策略**：
- 解釋技術限制或成本考量
- 提供替代方案
- 說明對專案時程的影響

**話術**：
```
「您提到的功能確實可以實現，但需要客製開發，
預估額外費用約 NT$ XX,XXX，時程延長 2-3 週。

另一個做法是使用現有外掛達到類似效果，
成本較低且可如期完成。您覺得如何？」
```

#### 案例 3：客戶遲遲不提供素材

**策略**：
- 記錄所有催素材的溝通
- 明確告知對時程的影響
- 必要時啟動合約延期條款

**Email**：
```
「由於素材尚未提供，專案時程需順延。
目前預估完成日期為 X 月 X 日（原訂 Y 月 Y 日）。

為確保專案品質，建議我們重新確認交付時程，
避免影響您的上線計劃。」
```

## 風險管理

### 常見風險與應對

| 風險 | 預防措施 | 發生時處理 |
|------|----------|------------|
| 需求範圍蔓延 | 合約明確定義範圍 | 額外需求另行報價 |
| 客戶素材延遲 | 設定素材提供截止日 | 時程順延並發 Email 確認 |
| 第三方服務問題 | 事先測試整合 | 提供替代方案 |
| 驗收標準爭議 | 驗收標準寫入合約 | 依合約條款處理 |

### 專案暫停/取消處理

**已收款項處理**：
- 依已完成工作比例計算
- 扣除已發生成本
- 保留設計稿智慧財產權（未付清前）
